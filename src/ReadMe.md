# Concolic Executor - RUST + Z3

## Overview

This is a concolic executor written in RUST with Z3

This program attempts to solve:
- `x^2 + y^2 = z^2`
- Print out the solution for `x`, `y`, `z`

## Requirement

1. Requires z3 Library

## Instructions:
1. `cargo build --release`

2. `cargo run`
